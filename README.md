# ✔ 오키스크 개요  
2023 DSC 리빙랩 지원사업의 일원으로 시작한 "오키스크"는 고령층의 키오스크 사용 어려움을 해결하기 위한 프로젝트입니다!  
키오스크 UI의 다양함으로 발생하는 혼란을 막기 위해 표준 UI와 템플릿을 제작하였습니다.
